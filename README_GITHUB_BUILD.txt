VIRAL ANIME WATER REMINDER — NO ANDROID STUDIO NEEDED

1. Go to GitHub and create a new empty repository.
2. Upload ALL contents of this folder, including the hidden .github folder.
   Important: upload the CONTENTS, not this outer folder itself.
3. Commit the files to the main branch.
4. Open the Actions tab in your repository.
5. Select "Build Android APK".
6. If it has already started, wait for it to finish. Otherwise click "Run workflow".
7. Open the completed workflow run.
8. Under Artifacts, download:
   Viral-Anime-Water-Reminder-APK
9. Unzip that artifact and install app-debug.apk on your Android phone.
10. Android may ask you to allow installation from your browser/files app.
11. Open the app and tap:
    - Allow floating character
    - Start hourly reminders
    - Test floating reminder now

NOTE:
This package uses GitHub's hosted Android build environment. No Android Studio is required.
